# Vehicle Rental Management System

## Project Overview

The Vehicle Rental Management System is a console-based Java application developed using Object-Oriented Programming (OOP) principles.  
The system allows users to manage vehicle rentals by viewing available vehicles, renting vehicles, returning vehicles, and calculating rental fees.

This project demonstrates key OOP concepts including:

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
- Custom Exception Handling
- Generics

---

# System Features

The system provides the following functionality:

1. View available vehicles
2. Rent a vehicle
3. Return a vehicle
4. Calculate rental fees
5. Prevent renting unavailable vehicles
6. Display vehicle inventory with different vehicle types

Vehicles included in the system:

- Toyota Corolla
- Honda Civic
- BMW X5
- Mercedes Benz GLE
- Audi Q7
- Volkswagen Passat
- Yamaha R1
- Kawasaki Ninja
- Honda CBR
- Suzuki GSXR

# How to Run the Project

Follow the steps below to compile and run the project.

## Step 1: Navigate to the Project Folder

Open a terminal and go to the project directory.

``cd GROUP9``


---

## Step 2: Compile the Program

Run the following command to compile the project.


``javac -d out src/main/.java src/models/.java src/interfaces/.java src/services/.java src/exceptions/.java src/utils/.java``


This will compile all Java files and place them in the `out` directory.

---

## Step 3: Run the Program

Execute the following command:

``java -cp out main.Main``
